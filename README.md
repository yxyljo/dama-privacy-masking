## 打码样式不止马赛克

很多人以为打码就是马赛克糊一块，DAMA 给的选择其实挺多：

- **色块**：纯色遮盖，干净利落，默认绿色可以改颜色
- **像素化**：经典马赛克效果
- **模糊**：高斯模糊，看起来比较自然
- **马克笔**：像用记号笔划掉一样
- **隐藏文字**：如果背景近乎纯色，可以把文字直接"消失"掉，完全看不出打码痕迹

最后这个"隐藏文字"功能挺厉害的，发出去的图根本看不出来遮过什么。

AI 不是万能的，有时候你想打码的部分它没识别出来。这时候 DAMA 也提供了几种手动方式：

- **拖拽矩形框**：手动框选任意区域

自动 + 手动结合，基本没有漏网之鱼。
